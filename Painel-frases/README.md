# Projeto Painel de Frases Motivacionais
O Painel de Frases Motivacionais é um projeto desenvolvido para inspirar o usuário diariamente. Ele exibe frases motivadoras de forma simples, dinâmica e agradável, ajudando a melhorar o foco, o humor e a produtividade. O projeto resolve problemas como falta de motivação, dificuldade de manter constância nos estudos ou trabalho e necessidade de estímulos positivos ao longo do dia. Assim, funciona como um espaço rápido e acessível para buscar encorajamento sempre que precisar.

## Funcionalidades 
-Exibição de frases motivacionais aleatórias
-Botão para gerar uma nova frase
- Interface limpa e amigável
-Possibilidade de adicionar frases novas
-Atualização das frases exibidas sem recarregar a página
-Organização simples para facilitar o uso diário

## Tecnologias ultilizadas
-HTML
-CSS
-JavaScript
-GitHub
-VS Code

## Como rodar
-Acesse o repositório do projeto no GitHub.
-Clique em Code e depois em Download ZIP para baixar todos os arquivos.
-Extraia a pasta em qualquer lugar do computador.
-Abra o VS Code (ou qualquer editor de sua preferência)
-No VS Code, vá em File > Open Folder e selecione a pasta do projeto.
-Abra o arquivo index.html no navegador clicando duas vezes nele ou usando uma extensão de Live Server (se tiver instalada).
-O painel será carregado no navegador, pronto para gerar frases motivacionais.

## O que aprendi
Durante o desenvolvimento, aprendi a manipular elementos da página com JavaScript, atualizar conteúdo sem recarregar a tela e organizar dados simples dentro do código. Também melhorei minhas habilidades com HTML e CSS para criar uma interface mais clara e agradável. Além disso, passei a entender melhor como usar GitHub para versionamento e como trabalhar de forma mais eficiente no VS Code.
